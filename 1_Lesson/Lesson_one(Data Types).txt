Типы данных в Python:
1. None (Неопределенное значение переменной)
2. Boolean Type (Логические переменные), True, False
3. Числа (Numeric Type):
	a. int - целые числа
	b. float - числа с плавающей точкой
	с. complex -  комплексные числа
# В питоне - все объекты!
4. Списки(Sequence Type)
	a. list - список (упорядочные по порядку элементы), (a = [1, 2, 3])
	b. tuple - кортеж (неизменяемый), (a = (1,2,3)
	c. range - диапазон, a = range(1, 10), list(range(10), list(range(1 <- от, 10, <- до, 3 <- шаг)
5. Строки(String Type)
	a. str - строка, можно обращаться как к списку, не изменяемые
6. Бинарные списки (Binary Sequence Types)
	a. bytes - байты
	b. bytearray - массив байт
	c. memoryview 
7. Множества
	a. Set - множество, a = [2, 5, 4, 6, 2], b = set(a), >>> {2, 4, 5, 6}, изменим, не поддерживает индексы
	b. Frozzenset- не изменяемый
8. Словари
	а. dict - словарь, a = {'a': 2, 'b': 12, 'c': 62}, изменяемые, не поддерживает индексы, доступ по ключам
